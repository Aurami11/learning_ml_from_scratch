# Rebuilding ML from Scratch

I’m using this repository to rebuild some machine learning and deep learning tools from scratch, mainly to understand better how they work behind the scenes.

I want this repo to stay flexible. The point is not to build a perfect framework, but to learn by implementing things myself, step by step.

The core path is:

- linear regression
- logistic regression
- backpropagation and simple neural nets
- attention
- a small GPT-like model based on transformers

After that, I also want to explore and rebuild other important blocks like:

- RNN
- CNN
- LSTM

So this repo will probably evolve over time.

It’s basically a mix of:
- learning notes
- from-scratch implementations
- small experiments
- and maybe a mini homemade ML/DL package by the end

Good luck to me :)
